# 문제 풀이 상태

* 다시 풀어야 할 문제: `AGAIN`
* 다시 풀지 않아도 될 문제: `PASS`
* 실패한 문제: `FAILED`

|  **번호**  |  **이름**  | **풀이 횟수** | **상태** |              **비고**              |
|:--------:|:--------:|:---------:|:------:|:--------------------------------:|
| **1961** | 숫자 배열 회전 |     1     | AGAIN  | 회전에 대한 규칙 찾기와 출력하는 부분이 까다롭다고 느꼈음 |
